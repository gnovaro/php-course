Basics

Object-oriented programming (OOP)

Class

A class is basically the definition or "blueprint" of how to build an object.

Example: 
        
            class Person
            {
                // Atributtes
                $name = null;
            }

Object

An object is an instance of a class, the construction of which is something concrete.

        
            class Person
            {

            }

            $john = new Person();

Attributes

The attributes of an object or its characteristics are what define it in itself: shape, color, height, width, weight, wheels

        
            class Person
            {
                // Attributes
                $name = null;
                $dob = null; // Date of birth
                $height = null;
                $weight = null;
            }

Methods

They are the actions they can take. In a car it could be starting, braking, stopping ...

        
            class Person
            {
                // Attributes
                $dob = null; // Date of birth
                $age = null;

                // Methods
                function getAge()
                {
                    // Today - Date of birth
                    return $age;
                }
            }


Constructor

The default constructor in php is the __construct() method, it is not mandatory to define it.

The constructor is the first method that is executed when building an object, it defines how it should be built, it is usually used to assign values to attributes of an object

        
        class Person
        {
            $name = null;

            function __construct($name = null)
            {
                $this->name = $name;
            }
        }

Here we are going to introduce a new keyword "this", which refers to itself

        
            $john = new Person('John');

Here we create an instance of the Person object by calling the constructor.
