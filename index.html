Introduction

php is an interpreted language created in 1994 by Rasmus Lerdorf, he wanted to know how many people were reading the curriculum vitae on his website and for this he created a CGI in Perl that showed the statistical result on the page itself. Lerdorf called that PHP script an acronym for Personal Home Page.

Download latest version from official site php.net

The latest version is 7.4 and you can download it from this link php 7.4 thread safe

  2. Add php dir to global PATH variable

Syntax

The php code is enclosed within the tags:

            
                <?php
                    // This is a comment

                    /*
                        This is a comment
                        in multiple
                        lines
                    */
                ?>

Note: If you are going to include pure php code we recommend using only the opening tag <?php The php interpreter ignores the closing tag
